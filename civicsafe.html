CivicSafe

Empowering communities through anonymous incident reporting and AI-powered safety insights

1 (Solo Project)
Full-Stack Web App

Project Overview

CivicSafe is a comprehensive community safety platform designed to empower citizens to actively participate in making their neighborhoods safer. The application enables users to anonymously report safety incidents, share valuable safety tips, and access AI-powered assistance through the integrated Tara bot. By leveraging location-based services and interactive mapping, CivicSafe creates a centralized hub where community members can visualize safety concerns in real-time, fostering collaborative efforts between citizens and local authorities to address safety issues promptly and effectively.

Technologies Used

Node.js Express.js MongoDB Leaflet.js JavaScript EJS-Mate Ollama AI HTML5 CSS3 MongoDB Compass

Key Features

  • Anonymous Incident Reporting: Users can report safety incidents, suspicious activities, or community concerns completely anonymously, protecting their identity while contributing to community safety without fear of retaliation.
  • Interactive Map Visualization: Integrated Leaflet.js mapping system displays all reported incidents and safety tips on an interactive map interface, allowing users to visualize safety hotspots and trends in their area at a glance.
  • Safety Tips Sharing: Community members can share safety tips, best practices, and helpful advice with neighbors, creating a collaborative knowledge base that promotes safer living environments.
  • Location-Based Intelligence: Advanced geolocation features enable precise incident reporting with exact coordinates, making it easier for authorities to respond and for citizens to stay informed about safety concerns in specific areas.
  • AI-Powered Assistant (Tara Bot): Integrated with local Ollama AI instance providing intelligent assistance, answering safety-related questions, offering guidance on incident reporting, and potentially analyzing patterns to generate safety insights.

Tara Bot - AI Assistant

CivicSafe integrates with a local Ollama AI instance to power Tara Bot, an intelligent assistant designed to enhance user experience. Tara can answer safety-related questions, provide guidance on proper incident reporting, suggest preventive measures, and analyze patterns in reported data to offer community-specific safety insights. This AI integration positions CivicSafe for future enhancements including predictive analytics, automated incident categorization, and personalized safety recommendations.

Technical Implementation

  • Backend Architecture: Built on Node.js and Express.js providing a robust, scalable server-side framework with efficient routing, middleware management, and API endpoint handling for seamless data flow.
  • Database Management: MongoDB serves as the NoSQL database solution with MongoDB Compass for visual data management, storing incident reports, safety tips, user sessions, and geolocation data with flexible schema design.
  • Mapping Integration: Leaflet.js implementation provides interactive, lightweight mapping capabilities with custom markers for different incident types, clustering for dense areas, and smooth pan/zoom functionality.
